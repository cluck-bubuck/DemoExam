#ReadMe файл для приложения ООО "Спортивный магазин"

##БД 
* создается с помощью скрипта "создание бд"
* для вставки данных, вроде бы, можно использовать файл "вроде как все данные здесь.bacpac"
###Для вставки данных нужно:
- Создать БД с помощью скрипта
- кликнуть ПКМ по БД и выбрать "Импорт приложения уровня данных"
- выбрать для загрузки файл "вроде как все данные здесь.bacpac"

##Приложение
* Может быть потребуется заменить строку подключения
